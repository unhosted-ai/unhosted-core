# Third-party notices

Unhosted is itself licensed under the [GNU Affero General Public License v3.0 or later](LICENSE). It builds on, links to, or ships alongside open-source software written by other people. This file lists the **direct dependencies** the project relies on, the license each is under, and where its source can be found. AGPL § 5 requires preserving these notices in distributions of the work.

This is not the exhaustive list of every transitive dependency — for a full build manifest, see [`Cargo.lock`](Cargo.lock) (Rust) and the various `package-lock.json` files in plugin repos. The full manifest of license texts for a given binary release ships in the release's `THIRD_PARTY_LICENSES.txt`, generated by `cargo about` at build time. (The build step that generates that file lands in a follow-up release; for current builds, refer to the upstream repositories named below.)

## How to extend this list

If you add a direct dependency, add a row. Transitive deps don't get rows — cargo, pip, and npm lock files are the source of truth for those. When generating release notes:

```bash
cargo install cargo-about --locked
cargo about generate about.hbs > THIRD_PARTY_LICENSES.txt
```

That command (and the `about.hbs` template the project ships when the build wiring lands) will produce a full per-release notice that satisfies AGPL § 5(a) for the binary.
