# Branding kit

Everything you need to make Unhosted look like Unhosted.

The runtime mark lives in [`/assets`](../assets) (used by the repo README). This folder is the **curated kit** — variations, social cards, palette, type specimens — the stuff a designer or contributor pulls from when making something new.

## What's here

```
branding/
├── lockup/                       horizontal + stacked + with-tagline marks
├── social/                       OG image, GitHub social preview, X header
├── colors/                       palette as CSS variables and swatches
├── icons/                        favicon variants for light / dark
└── README.md
```

## Use

Need a banner for a blog post? `social/og-image.svg`.
Updating the GitHub repo's social preview card? `social/github-social.svg` (drop into Settings → Social preview).
Adding a Twitter/X header? `social/x-banner.svg`.
Building a new page? Pull colors from `colors/palette.css`.

## Voice + rules

See [`../BRAND.md`](../BRAND.md). Don't override the rules there in a one-off design — push back instead.
